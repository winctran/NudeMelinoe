# Nude Melinoe

> NOTE: This mod requires some manual installation steps to get all its features

Hades II NSFW mod that resprites most of Melinoe in-game occurences with their nude versions. All sprites are original assets overpainted with no use of AI.

![Banner](images/banner.png)

## Features

Mod changes:

- Texture of 3D model
- 10 dialogue sprites 
- 2 bath dialogue sprites
- Boon select screen
- Main menu screen
- Hex select and upgrade screens

Mod does not change:

- 3D model geometry, as it is not possible with current state of modding tools
- Screens for weapon upgrade and select
- Biome introduction movies
- Ending movies
- Melinoe codex entry

## Install

- Use r2modman to install the mod with dependencies (disable NSFW filter)
- Get the movie files: In r2modman go `Settings > Browse Profile Folder > ReturnOfModding > plugins_data > Winctran-NudeMelinoe > Movies`
- Move the movies to game files: In r2modman check the path in `Change Hades II folder`. In this path navigate to `Hades II > Content Movies > 1080p` and move the `bik` and `bik_atlas` files

Make sure that in Hades 2 you use `high` setting for textures.
